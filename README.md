# 無心 - Mushin

何も考えず、ただ指を動かす。瞑想のためのちいさなゲームです。

疲れているとき、メンタルが落ちているときに、無心で手を動かして没頭するために作られています。スコアも失敗もタイマーもありません。

## 3つのモード

| モード | あそびかた |
|---|---|
| **泡** | 夜空をのぼる光の泡を、指でそっと割る。やわらかい鐘の音が響きます |
| **波** | 静かな水面に触れて、波紋を広げる。なぞると小さな波が続きます |
| **砂** | 枯山水の砂を、指でなぞって砂紋を描く。描いた模様はゆっくり消えていきます |

音はすべて端末内で合成されるので、通信は不要です(右上の ♪ で消音できます)。

## スマホアプリとして使う(PWA)

- **iPhone (Safari)**: 共有ボタン → 「ホーム画面に追加」
- **Android (Chrome)**: メニュー → 「ホーム画面に追加」(またはインストールの案内に従う)

ホーム画面から起動すると全画面のアプリとして動き、オフラインでも遊べます。

## 公開のしかた(初回のみ)

1. このブランチを `main` にマージする
2. GitHub のリポジトリ設定 → **Settings → Pages → Build and deployment → Source** を **GitHub Actions** にする
3. `main` に push されると自動でデプロイされます

## 技術

- 依存ライブラリなしの素の HTML / Canvas / Web Audio API
- Service Worker によるオフラインキャッシュ
- ビルド不要。`index.html` を開くだけでローカルでも動きます
