# 無心 - Mushin

めのまえの単純作業だけに、こころをあずける。瞑想のためのちいさなゲームです。

疲れているとき、メンタルが落ちているときに、次々と流れてくる小さなタスクをひたすらこなすことで、ほかのことを考えられなくするために作られています。ゲームオーバーも制限時間もありません。間違えてもやわらかい音が鳴るだけです。

## あそびかた

画面上部に指示が出ます。こなすと鐘の音が鳴って、すぐ次のタスクが始まります。それを、ただ、くりかえします。

| タスク | やること |
|---|---|
| **ふれる** | 指定された色の光だけを、すべてタップする |
| **すべてふれる** | 画面の光をぜんぶタップする |
| **じゅんばん** | 1 から順番に数字の光をタップする |
| **なぞる** | 光の道を、指でたどる |
| **かさねる** | 光をドラッグして、同じ色の輪に重ねる |

10 タスクごとに、こなした累計がふわっと表示されます。

音はすべて端末内で合成されるので、通信は不要です(右上の ♪ で消音できます)。

## スマホアプリとして使う(PWA)

- **iPhone (Safari)**: 共有ボタン → 「ホーム画面に追加」
- **Android (Chrome)**: メニュー → 「ホーム画面に追加」(またはインストールの案内に従う)

ホーム画面から起動すると全画面のアプリとして動き、オフラインでも遊べます。

## 公開のしかた(初回のみ)

1. このブランチを `main` にマージする
2. GitHub のリポジトリ設定 → **Settings → Pages → Build and deployment → Source** を **GitHub Actions** にする
3. `main` に push されると自動でデプロイされます

## 技術

- 依存ライブラリなしの素の HTML / Canvas / Web Audio API
- Service Worker によるオフラインキャッシュ
- ビルド不要。`index.html` を開くだけでローカルでも動きます
